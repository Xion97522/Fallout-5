This repository is also set up for GitHub Pages using the `docs/` folder.

## How to use

1. Place your site ZIP file in the repository root, for example `site.zip`.
2. Run `./extract-zip.sh site.zip` to unzip the archive into `docs/`.
3. Commit and push the changes.

## GitHub Pages

The workflow in `.github/workflows/pages.yml` deploys the contents of `docs/` to GitHub Pages on every push to `main`.

If you want to host manually instead, go to the repository Settings → Pages and select `main` branch / `docs/` folder.
